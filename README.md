## 📖 О проекте

**Project Slip_Star** — это портфолио-визитка в стиле киберпанк, объединяющее мои лучшие проекты: веб-приложения, игры, ИИ-интерфейсы и дизайн-эксперименты.

Сайт выполнен в тёмной неоновой эстетике с интерактивными элементами, анимациями и эффектами, создающими атмосферу футуристического города.

---

## ✨ Возможности

### 🎨 Визуальные эффекты
- **Неоновые градиенты** — текст, рамки, тени
- **Glitch-эффекты** — сбои на заголовках
- **Scanline / CRT overlay** — имитация старого экрана
- **Particle background** — анимированные частицы с соединениями
- **Terminal typing animation** — печатающийся текст

### 🧭 Навигация и UX
- **Терминальная навигация** — в стиле хакерского интерфейса
- **Фильтры проектов** — по категориям (Games, AI, Tools, Web)
- **Поиск** — по названию, тегам, описанию
- **Прогресс-бар** — при скролле страницы
- **Back to top** — неоновая кнопка
