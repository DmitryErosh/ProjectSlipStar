📌 Задача

Создать сайт-портфолио в стиле киберпанк с неоновой эстетикой, тёмной темой и интерактивными эффектами. Главная цель — передать атмосферу футуристического города, используя только HTML, CSS и JavaScript.

🎨 Цветовая палитра и атмосфера

Для создания киберпанк-атмосферы я выбрал следующие элементы:

  • Тёмный фон — #0a0a0f с радиальными градиентами

  • Неоновый циан — #00ffc8

✨ Glitch-эффекты

Один из ключевых элементов киберпанк-эстетики — эффект глитча (сбоя изображения). Я реализовал его с помощью CSS-псевдоэлементов и анимаций.

📄 Код: Glitch-эффект на CSS ▼ Показать код 
/* ===== GLITCH ЭФФЕКТ ===== */
.glitch {
    position: relative;
    display: inline-block;
}

.glitch::before,
.glitch::after {
    content: attr(data-text);
    position: absolute;
    top: 0;
    left: 0;
    width: 100%;
    height: 100%;
    pointer-events: none;
}

.glitch::before {
    color: var(--neon-pink);
    animation: glitch1 2s infinite;
}

.glitch::after {
    color: var(--neon-cyan);
    animation: glitch2 2s infinite;
}

@keyframes glitch1 {
    0%, 90%, 100% { transform: translate(0); opacity: 0; }
    92% { transform: translate(-2px, 1px); opacity: 0.7; }
    94% { transform: translate(2px, -1px); opacity: 0.5; }
}

🎮 Particle Background

Фоновые частицы создают ощущение движения и глубины. Они соединены линиями, образуя подобие нейронной сети или городских огней.

📄 Код: Particle System ▼ Показать код 
// ===== PARTICLE BACKGROUND =====
class Particle {
    constructor() {
        this.x = Math.random() * canvas.width;
        this.y = Math.random() * canvas.height;
        this.size = Math.random() * 2 + 0.5;
        this.speedX = (Math.random() - 0.5) * 0.3;
        this.speedY = (Math.random() - 0.5) * 0.3;
        this.opacity = Math.random() * 0.3 + 0.1;
    }
    
    update() {
        this.x += this.speedX;
        this.y += this.speedY;
        if (this.x < 0 || this.x > canvas.width) this.speedX *= -1;
        if (this.y < 0 || this.y > canvas.height) this.speedY *= -1;
    }
    
    draw() {
        ctx.beginPath();
        ctx.arc(this.x, this.y, this.size, 0, Math.PI * 2);
        ctx.fillStyle = `rgba(0, 255, 200, ${this.opacity})`;
        ctx.fill();
    }
}

// Соединение частиц линиями
function connectParticles() {
    for (let i = 0; i < particles.length; i++) {
        for (let j = i + 1; j < particles.length; j++) {
            const dx = particles[i].x - particles[j].x;
            const dy = particles[i].y - particles[j].y;
            const dist = Math.sqrt(dx * dx + dy * dy);
            if (dist < 150) {
                ctx.beginPath();
                ctx.moveTo(particles[i].x, particles[i].y);
                ctx.lineTo(particles[j].x, particles[j].y);
                ctx.strokeStyle = `rgba(0, 255, 200, ${0.05 * (1 - dist / 150)})`;
                ctx.stroke();
            }
        }
    }
}
