🎮 Игровая механика

Классические правила Тетриса:

  • 7 фигур: I, O, T, S, Z, L, J
  • Случайный спавн фигур
  • Управление: стрелки (влево/вправо/вниз/вращение)
  • Hard drop — мгновенное падение фигуры (пробел)
  • Система уровней — скорость растёт с каждой 5-й очищенной линией
  • Подсчёт очков и рекорд в localStorage

💡 Неоновый стиль

Каждый блок в игре имеет:

  • Свечение (box-shadow) — неоновый эффект
  • Градиентную заливку — объёмный вид
  • Пульсирующую подсветку — блоки мерцают
  • Тёмный фон — с сеткой и эффектом глубины
📄 Код: Отрисовка неонового блока ▼ Показать код 
// ===== НЕОНОВАЯ ОТРИСОВКА БЛОКА =====
function drawBlock(col, row, color, isBoard) {
    const x = col * BLOCK_SIZE;
    const y = row * BLOCK_SIZE;
    const size = BLOCK_SIZE - 1;

    // Свечение
    ctx.shadowColor = color;
    ctx.shadowBlur = isBoard ? 18 : 30;
    ctx.fillStyle = color;
    ctx.fillRect(x, y, size, size);

    // Внутренний блик (градиент)
    ctx.shadowBlur = 0;
    const grad = ctx.createRadialGradient(
        x+4, y+4, 2,
        x+10, y+10, 14
    );
    grad.addColorStop(0, '#ffffff88');
    grad.addColorStop(0.5, '#ffffff22');
    grad.addColorStop(1, '#ffffff00');
    ctx.fillStyle = grad;
    ctx.fillRect(x, y, size, size);

    // Рамка
    ctx.strokeStyle = '#ffffff44';
    ctx.lineWidth = 1;
    ctx.strokeRect(x, y, size, size);
}

🎯 Управление и UX

Игра поддерживает:

  • Клавиатуру — стрелки + пробел + P (пауза)
  • Мобильные устройства — сенсорные кнопки
  • Паузу — по клавише P или кнопке
  • Сохранение рекорда — в localStorage

// ===== ОБРАБОТКА КЛАВИШ =====
window.addEventListener('keydown', (e) => {
    const key = e.key;
    if (['ArrowUp','ArrowDown','ArrowLeft','ArrowRight',' '].includes(key)) {
        e.preventDefault();
    }
    if (gameOver || paused) {
        if (key === 'p' || key === 'P') togglePause();
        return;
    }
    switch(key) {
        case 'ArrowLeft': movePiece(-1, 0); break;
        case 'ArrowRight': movePiece(1, 0); break;
        case 'ArrowDown': movePiece(0, 1); break;
        case 'ArrowUp': rotateCurrent(); break;
        case ' ': hardDrop(); break;
        case 'p': togglePause(); break;
    }
});
